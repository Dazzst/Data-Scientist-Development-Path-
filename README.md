# Data-Scientist-Development-Path-
重新整理规划所有关于数据科学的知识点以方便复习。


马上要回国了，我需要复习所有已经学到的知识和技能，并且计划以后可能会接触并更新的技能。

内容将包含数据分析，商业分析，统计学，可视化数据处理，机器学习，神经网络。

本仓库包含成为数据科学家/数据分析师的学习路线与知识点整理。
