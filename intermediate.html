LEVEL 2 · INTERMEDIATE DATA ANALYSIS

Intermediate Data Analysis
从分析走向建模与工程实践

这一层聚焦经典机器学习、深度学习基础、数据工程和模型部署，让你从「分析师」切换到「会建模、会落地」的数据科学角色。

经典机器学习

从传统模型入手，理解监督学习和无监督学习的核心思想与评估方式。

  • 线性/逻辑回归、决策树、SVM 等模型理论与假设。
  • 模型评估与选择：Accuracy、Precision、Recall、F1、AUC、交叉验证。
  • 集成学习：Random Forest、GBDT、XGBoost、LightGBM。
  • 无监督学习：K-Means、DBSCAN 聚类，PCA、t-SNE 降维。

神经网络与深度学习基础

掌握基础神经网络结构与训练流程，为后续 NLP / CV / 推荐打基础。

  • 核心概念：神经元、激活函数、损失函数、梯度下降。
  • 常见结构：FNN、CNN、RNN / LSTM / GRU 的典型应用。
  • 框架入门：TensorFlow / Keras 与 PyTorch 的基本使用。
  • 在公开数据集上完成一个简单的分类/回归模型实验。

数据工程与存储基础

了解大数据生态与数据管道，为面向生产的系统化实践做准备。

  • 大数据生态：Hadoop、Spark 的角色与典型使用场景。
  • Spark 编程模型：RDD 与 DataFrame 的基本概念与常见操作。
  • 数据仓库与数据湖：ETL / ELT 流程与分层设计的直觉。
  • 云平台概览：AWS / Azure / GCP 中与数据相关的核心服务。

模型部署与 MLOps 入门

让模型走出 Notebook，形成可访问、可维护的线上服务雏形。

  • 模型持久化：pickle、ONNX 等格式及使用注意事项。
  • 服务化：使用 Flask / FastAPI / Streamlit 暴露推理接口或小应用。
  • 容器化：用 Docker 打包环境与服务，支持跨环境部署。
  • 初步理解 MLOps：数据→训练→部署→监控 的 Pipeline 思路。
