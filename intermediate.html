LEVEL 2 · INTERMEDIATE

Intermediate Data Analysis

这一层聚焦经典机器学习、深度学习基础、数据工程与模型部署，让你从分析走向建模和落地。

经典机器学习

从传统算法入手，理解分类、回归、聚类和降维的核心思想与评估指标。

  • 分类与回归：支持向量机（SVM）及其他基础模型。
  • 树模型与集成：Random Forest、GBDT、XGBoost、LightGBM。
  • 聚类与降维：K-Means、DBSCAN、PCA、t-SNE 等。
  • 模型评估：Accuracy、Precision、Recall、F1、AUC 等指标。

神经网络与深度学习基础

搭建和训练基础神经网络，理解深度学习的核心构件与常见网络结构。

  • 基础神经网络：全连接网络（FNN），前向传播与反向传播的直觉理解。
  • 常见结构：CNN、RNN、LSTM、GRU 的典型应用场景。
  • 框架入门：TensorFlow / Keras 与 PyTorch 的基本使用流程。
  • 简单实践：在公开数据集上训练一个分类/回归模型并评估效果。

数据工程与存储基础

理解数据工程的基本组件，为大规模数据处理和生产环境做准备。

  • 大数据框架：Hadoop 与 Spark 的核心概念与典型使用场景。
  • Spark 编程模型：RDD 与 DataFrame 的区别与常见转换操作。
  • 数据管道：ETL 与 ELT 的流程设计与实践要点。
  • 云平台概览：AWS / Azure / GCP 中常见的数据与计算服务。

模型部署与 MLOps 入门

把模型从 Notebook 带到线上，理解端到端机器学习系统的基本形态。

  • 模型保存与格式：pickle、ONNX 等常见模型持久化方式。
  • 服务化框架：用 Flask / FastAPI / Streamlit 包装模型为 API 或应用。
  • 容器化：用 Docker 打包环境与服务，方便部署和迁移。
  • MLOps Pipeline 概念：数据 → 训练 → 部署 → 监控 的闭环思路。
