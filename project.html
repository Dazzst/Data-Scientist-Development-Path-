Projects & Kaggle
用项目把所有知识串起来

这一部分把前面各层的技能落地到真实项目和竞赛，从端到端分析到完整 ML 项目，再到专项方向实战。

Kaggle 实战路径

从入门到进阶，利用 Kaggle 作为实战练兵场。

  • 阅读《Kaggle_handbook》：竞赛流程、提交、Leaderboard 心态。
  • 从 Tabular 任务起步，熟悉特征工程与树模型调参。
  • 逐步尝试 NLP / CV 竞赛，接触不同模态与评估方式。
  • 参与讨论区与分享 Notebook，学习顶尖方案。

典型公开数据集项目

利用经典公开数据集构建具有复用价值的项目模板。

  • Brazilian E-Commerce Public Dataset by Olist：电商业务与用户行为分析。
  • 端到端流程：问题定义 → 数据理解 → 特征工程 → 建模 → 业务解读。
  • 为每个项目写 README 和报告，强调「业务故事」。
  • 在 GitHub 上维护一个项目合集仓库，形成作品集。

分层项目建议

按基础 / 中级 / 高级设计自己的项目路线。

  • 基础：完整的数据分析报告（EDA + 可视化 + 结论）。
  • 中级：经典 ML 项目（回归 / 分类）+ 模型评估与对比。
  • 高级：专项方向项目（推荐、NLP、CV、时序等）。
  • 每个阶段至少完成 1–2 个项目作为里程碑。
