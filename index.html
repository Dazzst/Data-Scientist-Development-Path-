Level 1 · Basic

数据分析基础（Basic Data Analysis） 目标：能看懂 & 模仿完成基础分析

这一阶段主要是打基础：熟悉 Python 环境、掌握常用数据结构和语法，能用 Pandas 读写数据并做最基本的统计与可视化。

建议学习内容

  • 开发环境：Anaconda / Python、VS Code / Jupyter、Git 基本操作。
  • Python 基础语法：变量、条件与循环、函数、列表 / 字典等数据结构。
  • Pandas 入门：读取 CSV / Excel、查看数据、基本清洗（缺失值、去重等）。
  • 基础可视化：Matplotlib / Seaborn 绘制折线图、柱状图、散点图。

阶段产出示例

  • 1～2 个 Notebook，对公开数据集做描述性分析（均值、中位数、分布等）。
  • 一份个人速查表：记录常用 Pandas 操作和可视化代码模板。
Level 2 · Intermediate

数据分析中级（Intermediate Data Analysis） 目标：能独立完成小型分析项目

这一阶段重点从「写代码」走向「做分析」，学会围绕一个问题组织数据、指标和图表，并把结论讲清楚。

建议学习内容

  • 数据清洗进阶：缺失值处理、异常值检测、类别编码、时间序列处理。
  • 探索式数据分析（EDA）：分组统计、透视表、相关性分析。
  • 业务指标意识：转化率、留存、漏斗、分层等基本分析框架。
  • 报告输出：用图表 + 文字写一份完整的分析报告或文章。

阶段产出示例

  • 2 个以上端到端 EDA 项目（包含：问题、分析过程、结论）。
  • 至少一篇对外可分享的分析总结（博客 / README / Notion 页面）。
Level 3 · Advanced

数据分析高级（Advanced Data Analysis） 目标：系统使用统计 & 建模方法

在这一阶段，你会把统计学和简单建模系统地引入分析流程，开始关注因果、实验设计和更严谨的结论。

建议学习内容

  • 统计基础：估计与置信区间、假设检验、p 值、方差分析等。
  • 回归与分类模型：线性回归、逻辑回归、简单树模型。
  • 实验设计：AB 实验、分层、样本量、实验指标设计。
  • 多维分析：分群分析、聚类、主成分分析等方法。

阶段产出示例

  • 1～2 个结合统计模型的分析案例（例如：回归 + 业务解释）。
  • 一份「分析模板」：从问题定义到结论呈现的一整套步骤规范。
Level 4 · Expert

数据分析大神（Expert Level） 目标：方法论 & 影响力

这一阶段更多是「横向拉通」：你不只是完成分析，而是能设计整体方案、建立分析框架并教会别人怎么做。

建议发展方向

  • 构建分析框架：针对常见问题形成可复用的分析路径和模板。
  • 跨团队沟通：用非技术语言向业务方解释数据和模型。
  • 复盘与沉淀：把项目经验总结成文档、内部分享、公开输出。
  • 指导新人：帮助别人设计学习路径和项目练习。

阶段产出示例

  • 一份系统的「数据分析手册」或系列文章 / 视频。
  • 在团队或社区中，持续稳定的分享与答疑记录。
Practice

数据实战（Projects & Kaggle） 目标：把能力落到作品集

项目是把前面所有阶段的知识真正串起来的地方，你可以结合 Kaggle、真实业务和个人兴趣，设计 3～5 个有深度的项目。

项目类型建议

  • 真实业务或模拟业务项目：从问题 → 数据 → 分析 → 结论 → 建议。
  • Kaggle / 公开数据竞赛：选择少量题目深挖，而不是大量浅尝辄止。
  • 个人兴趣项目：例如心理学视频 / 文本、教育数据、内容推荐等。

项目结构建议

  • README 中写清：问题背景、数据来源、分析步骤、可复现代码。
  • 每个项目至少包含：数据清洗、EDA、模型或统计方法、结论与反思。

建议在仓库里为每个项目建立单独文件夹，并在这个页面上列出链接，方便浏览和展示。

Resources

学习资料（Resources） 目标：少而精、可维护

这里可以集中整理你在不同阶段用过、觉得值得推荐的书籍、课程、视频和博客，并标注适用阶段和你的备注。

资料整理建议

  • 按阶段分组：基础 / 中级 / 高级 / 实战 / 思维。
  • 每条资源写上：类型（书 / 课）、难度、适合阶段、你的一句话评价。
  • 可以在仓库中单独建一个 resources.md 详细列出链接，并从这里跳转。

示例分类

  • 书籍：Python、统计学、机器学习、商业分析等。
  • 在线课程：系统课程 + 碎片化视频（YouTube / B 站）组合。
  • 工具与模板：你常用的 Notebook 模板、代码片段合集等。
