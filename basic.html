LEVEL 1 · BASIC DATA ANALYSIS

Basic Data Analysis
从环境、Python 到 Pandas 的系统入门

这一层的目标是搭好开发环境，打牢 Python 和数据分析工具的基础，完成至少一个端到端的小型数据分析项目。

编程工具与环境

安装配置 Python 环境，熟悉日常开发工具和项目目录结构，给后续所有阶段打地基。

  • Anaconda / Miniconda 安装与虚拟环境管理。
  • VS Code 与 Jupyter Notebook / Lab 的基本使用。
  • 约定统一的项目结构：data / notebooks / src / outputs。
  • 用 Git 管理练习项目，并托管到 GitHub。

Python 语言基础

用代码表达你的思路，写得出小脚本，读得懂示例代码。

  • 基础类型与容器：数字、字符串、list、dict、set。
  • 控制流程：if / for / while，列表推导式。
  • 函数与模块：参数、返回值、标准库的基本使用。
  • 文件读写与简单异常处理，读写 CSV 文本。

Numpy 与 Pandas

学会真正用 Pandas 和 Numpy 操作数据，是数据分析的核心工作台。

  • Numpy ndarray：创建、索引、切片和基本聚合。
  • Pandas Series / DataFrame：读取 CSV / Excel，查看概况。
  • 数据清洗：缺失值、重复值、类型转换和新特征构造。
  • groupby、pivot_table 做汇总统计和业务指标分析。

可视化与数据故事

用合适的图表和结构化叙事，把结论讲清楚，而不是只给一堆表。

  • Matplotlib / Seaborn 画常用图：折线、柱状、散点、直方图。
  • 图表美化：标题、坐标轴、图例、子图布局与配色。
  • 结构化报告：问题 → 分析过程 → 发现 → 结论与建议。
  • 完成一个端到端小项目作为 Level 1 里程碑。
