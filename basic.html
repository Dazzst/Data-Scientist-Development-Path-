LEVEL 1 · 数据分析基础（Basic Data Analysis）

本阶段目标：从零搭建数据分析环境，掌握 Python 语言基础与常用数据分析工具， 能够独立完成一个小型数据分析项目并产出可阅读的报告。

一、环境与工程基础（Environment & Engineering）

1.1 Python 发行版与虚拟环境

  • 推荐安装 Anaconda 或 Miniconda，统一管理 Python 版本与第三方库。
  • 学会使用 conda 或 pip 创建、激活、删除虚拟环境。
  • 理解 requirements.txt 和 environment.yml 的作用，用于导出和复现环境。

1.2 开发工具与运行方式

  • VS Code：安装 Python 插件，配置解释器，运行 .py 文件，设置断点进行调试。
  • Jupyter Notebook / JupyterLab：创建 notebook，分单元格编写、运行代码与说明文字。
  • 命令行运行脚本：在终端中使用 python main.py 执行项目入口文件。

1.3 项目结构与版本控制

  • 基础目录结构示例： 
    project/
  data/
    raw/        # 原始数据
    processed/  # 处理后的数据
  notebooks/    # 探索性分析与实验
  src/          # 可复用脚本与函数
  outputs/      # 图表、报告等输出
  README.md     # 项目说明
  • 使用 Git 做版本控制：git init、git add、git commit、git log 等基础命令。
  • 将项目托管到 GitHub，养成小步提交、写清 commit message 的工程习惯。

二、Python 语言基础（Python Basics）

本小节目标：掌握 Python 核心语法，能够编写简单脚本，为后续数据分析打基础。

2.1 核心数据类型与容器

  • 基本类型：int、float、str、bool。
  • 容器类型：list、tuple、dict、set，掌握增删改查和遍历。
  • 常用内置函数：len、type、sorted、range 等。

2.2 控制流程

  • 条件语句：if / elif / else，根据条件执行不同分支。
  • 循环语句：for、while，配合 break、continue 使用。
  • 列表推导式：用一行代码完成「循环 + 条件 + 构造新列表」。

2.3 函数与模块

  • 函数定义：参数、返回值、默认参数、关键字参数，理解「封装」思想。
  • 模块与包：import、from ... import ... 的基本用法。
  • 常用标准库：os、pathlib、datetime 等在数据分析项目中的简单应用。

2.4 文件读写与异常处理

  • 使用 open 读写文本文件，理解文本模式与二进制模式的区别。
  • 用 Python 读取和写入 CSV 文件（不依赖第三方库的基础方式）。
  • 简单异常处理：try / except，提升脚本的稳定性和可维护性。

三、Numpy 与 Pandas 数据分析

本小节目标：掌握 Numpy 与 Pandas 的核心用法，能够完成数据加载、清洗、转换与基础分析。

3.1 Numpy 基础

  • 创建 ndarray：np.array、np.zeros、np.ones、np.arange 等。
  • 数组属性：shape、dtype，基本索引与切片。
  • 常见运算：元素级加减乘除、聚合函数（均值、标准差、最大值、最小值）以及基本矩阵运算。

3.2 Pandas 数据结构

  • Series 与 DataFrame 的概念与区别。
  • 数据读取：read_csv、read_excel，了解 read_sql 的存在。
  • 数据浏览：head、tail、info、describe、value_counts。

3.3 数据清洗与特征构造

  • 缺失值处理：isna / notna、fillna、dropna。
  • 重复值处理：drop_duplicates，理解唯一键的概念。
  • 类型转换：astype，字符串列转时间 to_datetime 等。
  • 新特征构造：基于运算或条件创建新列，使用 loc 按条件赋值。

3.4 分组、透视与合并

  • 分组聚合：groupby + sum/mean/count 等。
  • 透视表：pivot_table、crosstab 实现多维统计。
  • 多表合并：merge、join，理解 inner / left / right / outer 等连接方式。

四、数据可视化与结果输出

本小节目标：用图表直观表达分析结论，并将结果导出为可分享的形式。

4.1 基础图表类型

  • Matplotlib：折线图、柱状图、散点图、直方图。
  • Seaborn：countplot、boxplot、heatmap 等常用统计图。
  • 根据数据类型和问题选择合适图表，避免误导性可视化。

4.2 图表美化与布局

  • 设置标题、坐标轴标签、图例、刻度和颜色。
  • 使用子图（subplot）展示多个相关图表，讲清一个完整问题。
  • 控制图像尺寸、分辨率，为报告或 PPT 准备高质量图像。

4.3 报告与导出

  • 在 Notebook 中用 Markdown 编写分析过程与结论，形成带图表的故事化报告。
  • 导出结果：将关键表格保存为 CSV/Excel，将图像保存为 PNG/SVG。
  • 为未来复现预留信息：记录数据来源、处理步骤、关键参数。

五、小项目与阶段产出（Milestone）

本小节目标：通过一个端到端的小项目，把本阶段的知识串起来，形成可展示的成果。

5.1 小项目要求

  • 选择至少一个公开数据集，例如：销售数据、用户行为数据、简单 A/B 测试数据等。
  • 完整流程：数据加载 → 清洗 → 分析 → 可视化 → 结论与建议。
  • 在项目中至少使用一次 groupby、一次透视表和若干图表。

5.2 工程化实践

  • 按照约定目录结构组织代码、数据和输出。
  • 用 Git 记录关键阶段的提交，例如：完成清洗、完成 EDA、完成可视化。
  • 编写 README.md，说明项目背景、数据来源、主要问题、方法和结论。

5.3 进阶指引

  • 如果顺利完成本阶段，可以继续学习：SQL、统计学基础、可视化/BI 工具（如 Power BI、Tableau）。
  • 为后续机器学习、实验设计等主题打下数据与工程基础。
