数据分析基础（Basic Data Analysis）

这一阶段的核心，是熟悉 Python、Pandas 和基础可视化工具，能够在 Notebook 里完成一个完整的「从数据加载到简单结论」的流程。

一、环境与工具

  • 安装并熟悉 Anaconda / Python、VS Code 或 Jupyter Notebook。
  • 学会创建虚拟环境、安装常用包（如 pandas、numpy、matplotlib）。
  • 掌握 Git 的最基础用法：初始化仓库、提交、推送到 GitHub。

二、Python 语言基础

  • 变量、数值、字符串、列表、字典等基础数据类型。
  • 条件判断与循环（if / for / while）。
  • 函数的定义与调用，模块与包的基本概念。

三、Pandas 数据分析入门

  • 读取数据：CSV / Excel / SQL 到 DataFrame。
  • 查看与选择：head()、info()、describe()、列选择与行筛选。
  • 数据清洗：处理缺失值、去重、类型转换、简单派生字段。

四、基础可视化

  • 使用 Matplotlib / Seaborn 画折线图、柱状图、散点图、直方图。
  • 学会设置标题、坐标轴标签、图例等基本元素。
  • 尝试用 2～3 个图同时讲清一个问题。

五、阶段小结与产出

  • 完成 1～2 个公开数据集的小项目（例如：简单的销售数据 / 用户行为数据）。
  • 整理出你的第一个「基础分析模板」Notebook，后面阶段可以在此基础上扩展。

建议：在 GitHub 仓库中为本阶段建立一个 basic/ 文件夹，放基础项目和 Notebook，并在 README 里简要说明练习内容。
